# Acciojob
# HireReady projects 
HireReady is a simple and responsive job portal frontend web application designed to help users get started with authentication-based UI layouts and modern CSS styling.  
This project focuses on clean design, reusable CSS variables, and responsive layouts.

## Features
1. Real-world UI layout
2. Responsive design
3. Clean HTML + CSS coding
4. Professional folder structure

## Technologies Used
1. HTML5
2. CSS3
3. CSS Grid
4. Responsive Design
